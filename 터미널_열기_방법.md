# 터미널 열기 방법 (Cursor/VS Code)

## 🎯 가장 쉬운 방법

### 방법 1: 하단 패널에서 TERMINAL 탭 클릭
1. 화면 **가장 아래쪽**을 보세요
2. "Plan, @ for context, / for commands"라고 써있는 패널이 보이나요?
3. 그 패널의 **위쪽**에 작은 탭들이 있을 겁니다:
   - TERMINAL ← 이걸 클릭하세요!
   - OUTPUT
   - PROBLEMS
   - 등등...

### 방법 2: 키보드 단축키
- **Windows**: `Ctrl + ~` (백틱 키, 숫자 1 왼쪽 키)
- 이 키를 누르면 터미널이 자동으로 열립니다!

---

## ✅ 터미널이 열리면 무엇을 봐야 하나요?

터미널이 열리면 이런 메시지가 보여야 합니다:

```
▲ Next.js 14.2.33
- Local:        http://localhost:3000
- Ready in 2.3s
```

이 메시지가 보이면 개발 서버가 실행 중입니다!

---

## 📝 Excel 다운로드 후 로그 확인하기

1. **터미널을 열어주세요** (위 방법 사용)
2. **관리자 페이지로 가서** "Excel 다운로드" 버튼 클릭
3. **터미널을 다시 보세요** - 자동으로 로그가 나타납니다!

터미널에 이런 로그들이 나타납니다:
```
[Export] 🔄 Attempt 1: Calling getResponsesBySurvey...
[Export] 🔍 Attempt 1 returned 55 responses
[DB] getResponsesBySurvey - Found 55 responses in Supabase query
```

---

## ❓ 여전히 터미널이 안 보이나요?

### 새 터미널 열기
1. 상단 메뉴에서 **"Terminal"** 클릭
2. **"New Terminal"** 선택
3. 새로 열린 터미널에서 `npm run dev` 입력

### 개발 서버 실행
터미널이 열리면 다음 명령어를 입력하세요:
```bash
npm run dev
```

이제 Excel 다운로드를 시도해보세요!
