# 로그 확인 간단 가이드 📝

## 🎯 가장 쉬운 방법

### 1. Cursor 하단에 "터미널" 탭이 보이나요?
   - 보인다면 → 2단계로 이동
   - 안 보인다면 → 키보드에서 `Ctrl + ~` (백틱) 누르기

### 2. 터미널에서 이것을 찾으세요:
   ```
   ▲ Next.js 14.2.33
   - Local:        http://localhost:3000
   ```
   이 메시지가 보인다면 개발 서버가 실행 중입니다! ✅

### 3. Excel 다운로드 버튼을 누르세요
   - 관리자 페이지에서 "Excel 다운로드" 클릭

### 4. 터미널을 바로 확인하세요
   - 터미널에 자동으로 로그가 나타납니다
   - 아래와 같은 메시지들이 보여야 합니다:
     ```
     [Export] 🔄 Attempt 1: Calling getResponsesBySurvey...
     [Export] 🔍 Attempt 1 returned 55 responses
     ```

---

## ❓ 여전히 모르겠다면

### 방법 A: 개발 서버 다시 시작
터미널에서 다음을 입력하세요:
```
npm run dev
```

그리고 Excel 다운로드를 다시 시도하세요!

### 방법 B: 스크린샷 찍기
터미널 화면을 스크린샷으로 찍어서 보여주시면 더 정확히 도와드릴 수 있습니다!

---

## 📋 확인해야 할 것들

Excel 다운로드 후 터미널에서 이 숫자들을 찾아주세요:

1. **몇 개의 응답이 조회되었나요?**
   - `returned 55 responses` ← 이 숫자
   
2. **몇 번 시도했나요?**
   - `Attempt 1`, `Attempt 2`, ... ← 몇 번까지?
   
3. **최종적으로 몇 개를 사용했나요?**
   - `Final: Using 55 responses` ← 이 숫자

이 숫자들을 알려주시면 문제를 해결할 수 있습니다!
