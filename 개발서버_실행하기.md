# 개발 서버 실행하기 🚀

## 개발 서버가 안 보이는 경우

터미널에 Next.js 메시지가 안 보인다면, 개발 서버가 실행되지 않은 것입니다.

## ✅ 해결 방법

### 1단계: 터미널 확인
- 터미널이 열려 있는지 확인하세요
- `Ctrl + ~` 키로 터미널 열기

### 2단계: 개발 서버 실행
터미널에서 다음 명령어를 입력하고 **Enter** 키를 누르세요:

```bash
npm run dev
```

### 3단계: 기다리기
- 몇 초 기다리면 다음과 같은 메시지가 나타납니다:

```
▲ Next.js 14.2.33
   - Local:        http://localhost:3000
   - Ready in 2.3s
```

### 4단계: 확인
- 이 메시지가 보이면 성공입니다! ✅
- 이제 개발 서버가 실행 중입니다

---

## 📝 중요한 점

1. **터미널을 닫지 마세요**
   - 터미널을 닫으면 개발 서버가 중지됩니다
   - 서버가 실행 중인 동안은 터미널을 열어두세요

2. **"Ready" 메시지를 확인하세요**
   - "Ready in X.Xs" 메시지가 나타날 때까지 기다리세요
   - 이 메시지가 나타나야 서버가 완전히 시작된 것입니다

3. **에러가 발생하면**
   - 터미널에 빨간색 에러 메시지가 나타날 수 있습니다
   - 에러 메시지를 복사해서 알려주시면 도와드리겠습니다

- 아무것도 나타나지 않나요?
